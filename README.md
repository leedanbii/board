# 게시판

---

# 🎯 기능 목록

## 1. 회원가입

- 사용자가 **아이디, 비밀번호, 이름, 생년월일**을 입력하면 회원가입을 한다.
- 이름은 최대 **5글자**로 제한된다.

**예외 상황**

- 이름이 5글자를 넘는 경우

---

## 2. 로그인

- 사용자가 **아이디, 비밀번호**로 로그인을 한다.

**예외 상황**

- 아이디와 비밀번호가 일치하지 않을 경우
- 아이디가 존재하지 않는 경우

---

## 3. 게시글 작성

- 로그인 한 사용자가 제목, 내용을 입력한다.
- 제목은 30자, 내용은 1000자로 제한된다.

**예외 상황**

- 로그인 하지 않은 사용자가 게시글 작성을 시도하는 경우
- 제목이 30자를 초과하는 경우
- 내용이 1000자를 초과하는 경우

---

## 4. 게시글 목록 / 조회

- 사용자가 작성한 게시글 목록을 보여준다.
